◆ updateList()

void ShortCutsViewModel::updateList ( std::string  searchInput)

Updates the list of shortcuts based on a search filter.

Filters the memory cells from the MemoryCellManager based on the search input. An empty input string matches all cells. It then resets the view and cell indices.

Parameters
searchInputThe string to search for within the content of the memory cells.
