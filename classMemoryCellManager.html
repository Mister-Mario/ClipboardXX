◆ moveContentsOneDown()

void MemoryCellManager::moveContentsOneDown ( )

Shifts the content of each memory cell down by one position.

This is used for the auto-copy feature, where the content of cell 0 moves to cell 1, cell 1 to cell 2, and so on, to make room for a new entry in cell 0.
