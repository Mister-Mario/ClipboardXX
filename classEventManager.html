Processes a generic RmlUi event.

Processes an event coming through from RmlUi.

Parameters
[in]eventThe RmlUi event that spawned the application event.

This is the primary entry point for handling events from the UI. It parses a semicolon-separated string of commands and processes each one individually.

Parameters
eventThe source event object from RmlUi.
valueThe string value from the 'data-event' attribute, containing commands.
