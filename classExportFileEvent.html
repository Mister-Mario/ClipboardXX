Handles the final step of the export process.

This function is triggered after the user has specified a file name and is ready to export. It validates that a delimiter has been provided and a file path has been set. If both are valid, it retrieves all memory cell contents, exports them to the specified file, and switches the UI back to the main window.

Parameters
eventThe RmlUi event that triggered this action, used to find UI elements.
valuesA list of string parameters (not used in this handler).
