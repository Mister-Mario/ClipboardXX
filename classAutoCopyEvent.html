AutoCopyEvent::AutoCopyEvent ( MemoryCellManager *  memoryCellManager)
explicit

Constructor for the AutoCopyEvent.

Sets up a permanent base callback on the ClipboardListener. When the auto-copy feature is active, this callback will automatically shift the existing clipboard history down to make room for the new item whenever a new copy action is detected system-wide.

Parameters
memoryCellManagerA pointer to the manager for clipboard memory cells.
