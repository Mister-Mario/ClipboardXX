KeyShortCut * KeyShortCutManager::FilterShortCuts ( Rml::Input::KeyIdentifier  key)

Filters the active shortcuts list based on a key press.

This function implements the logic for detecting multi-key shortcuts. It maintains an internal state of possible shortcuts and narrows them down with each key press.

Parameters
keyThe key identifier that was just pressed.
Returns

void KeyShortCutManager::LoadKeyShortCuts ( )
private

Loads custom key shortcuts from a configuration file.

Reads shortcuts from 'shortCuts.csv' and parses them, adding them to the base list. The format is expected to be two lines per memory cell (copy, then paste).

◆ SelectFile()

std::vector< std::string > KeyShortCutManager::SelectFile ( )
private

Selects the appropriate shortcut file to load.

It reads both a user-modifiable shortcut file and a base/default file. If the sizes differ (indicating a potential corruption or mismatch), it returns the base file's content as a fallback.

Returns
std::vector<std::string> A list of strings, each representing a shortcut definition.

◆ WriteShortCuts()

void KeyShortCutManager::WriteShortCuts ( )
private

Writes the current custom shortcuts to the configuration file.

It skips the first two base shortcuts (system copy/paste) and writes the rest to 'shortCuts.csv' for persistence.
