Handles the event to expand from the shortcuts menu to the main application view.

This function sets the currently highlighted shortcut as the active one, adjusts the main window's appearance (enables border, maximizes), and then switches the visible RmlUi document from the shortcuts menu to the main window.

Parameters
eventThe RmlUi event that triggered this action.
valuesA list of string parameters.
