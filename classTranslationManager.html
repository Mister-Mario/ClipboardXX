◆ loadLanguage()

bool TranslationManager::loadLanguage ( const std::string &  code)

Loads a language translation file.

This function attempts to load a JSON file corresponding to the provided language code. If the specific language file is not found, it falls back to a default (en_US.json). It then parses the JSON and flattens its structure into a key-value map (e.g., "header.title").

Parameters
codeThe language code to load (e.g., "en_US", "es_ES").
Returns
bool True if a language file was successfully loaded and parsed, false otherwise.
