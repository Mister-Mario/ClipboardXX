Rml::StringList  values

Performs the core logic of the copy event.

This method is called by the base WindowFocusEvent::handle method after focus has been set. It sets up a one-time callback on the ClipboardListener. This callback will capture the text from the next copy operation and save it directly into the specified memory cell. It then simulates a copy (Ctrl+C) command.

Parameters
eventThe RmlUi event that triggered this action.
valuesA list of string parameters, where values[1] is expected to be the index of the target memory cell.

Simulates a copy (Ctrl+C) keyboard shortcut.

Uses the Windows SendInput API to programmatically press and release the Left Control and 'C' keys, which triggers a copy action in the currently focused application.
