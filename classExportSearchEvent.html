Handles the event to initiate a file search for export.

This function calls the file manager to open a system "save as" dialog. The title of the dialog is retrieved from the translation manager to support localization.

Parameters
eventThe RmlUi event that triggered this action (not directly used).
valuesA list of string parameters (not used in this handler).
